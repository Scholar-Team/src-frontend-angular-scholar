Continue estudando com Scholar

Com o Scholar ficou mais fácil de fazer seus estudos no conforto da sua casa. Com uma interface simples e clara, continue seus estudos mesmo longe da escola. Tá esperando o quê? Venha para conhecer o Scholar.

Projeto Scholar

Projeto Scholar, criado durante o período do curso FullStack Java Web, um curso da Digital House, oferecido gratuitamente por meio do Becas, da Santander. O projeto consiste em uma plataforma voltada para o ensino remoto, trazendo uma interface mais amigável para os estudantes e professores, facilitando assim o processo de aprendizado.
